Урок 6: Остаток от деления, инкремент и декремент

Остаток от деления, инкремент и декремент в Dart

Операции присваивания

Одна из самых частых операций — увеличить или уменьшить значение переменной. Это можно сделать, изменив значение переменной и присвоив новое значение обратно в ту же переменную.

Dart — Базовое присваивание

int value = 42;

value = value + 7; // 49
value = value - 7; // 42

// value = 42 + 7 = 49
// value = 49 - 7 = 42

Такие операции встречаются настолько часто, что для них придумали сокращённые формы.

Dart — Сокращённые операции присваивания

num value = 42;

value += 7; // value = value + 7;
value -= 7; // value = value - 7;
value *= 7; // value = value * 7;
value /= 7; // value = value / 7;
value %= 7; // value = value % 7;

Инкремент и декремент

Инкремент и декремент

Крайне распространённые операции — увеличение/уменьшение переменной на единицу.

Dart — Инкремент и декремент

int value = 10;

// Можно так
value += 1;
value -= 1;

// Но удобнее так
value++; // инкремент
value--; // декремент

Инкремент ++ делает то же, что и += 1, но короче. Аналогично декремент --.

Важно

Операторы ++ и -- работают только при изменении на единицу.

Префиксная форма

Префиксная форма изменяет значение переменной перед её использованием. Префиксный инкремент возвращает уже увеличенное значение.

Dart — Префиксный инкремент (++a)

int a = 10;
int b = ++a;

print(a); // 11
print(b); // 11

Сначала a увеличивается на единицу, затем значение присваивается b. В итоге a и b равны 11.

Постфиксная форма

Постфиксная форма изменяет значение переменной после её использования. Постфиксный инкремент возвращает значение до увеличения.

Dart — Постфиксный инкремент (a++)

int a = 10;
int b = a++;

print(a); // 11
print(b); // 10

Сначала значение a присваивается b, затем a увеличивается на единицу. В итоге a = 11, b = 10.

Порядок приоритета

Понимание приоритета операторов важно для корректного чтения и написания выражений.

Сценарии использования % и условных конструкций

Деление нацело

Часто нужно проверить, делится ли число нацело. В этом помогает оператор остатка от деления %: если остаток равен 0, число делится нацело.

Dart — Проверка кратности 5

int value = 25;

if (value % 5 == 0) {
  print("Число $value кратно 5");
} else {
  print("Число $value на цело не делится на 5");
}

Dart — Кратно 3 и 5 одновременно

int value = 15;

if ((value % 5 == 0) && (value % 3 == 0)) {
  print("Число $value кратно 5 и 3 одновременно");
} else {
  print("Число $value на цело не делится на 5 и 3 одновременно");
}

Dart — Проверка чётности

int value = 42;

if (value % 2 == 0) {
  print("Число $value чётное");
} else {
  print("Число $value нечётное");
}

Получение числовых разрядов

Разбор числа на разряды

Единицы, десятки, сотни, тысячи и т.д.

Dart — Разбор числа на разряды

int value = 2354;

// Разбить число на разряды
print(value % 10);   // 4   Единицы
print(value % 100);  // 54  Десятки
print(value % 1000); // 354 Сотни
print(value);        // 2354 Тысячи

// Получить каждую цифру по отдельности
print((value / 1000).truncate());      // 2
print((value / 100).truncate() % 10);  // 3
print(((value % 100) / 10).truncate()); // 5
print(value % 10);                      // 4

Dart — Целочисленное деление (~/)

// Для целочисленного деления используйте оператор ~/ 
int value = 2354;

print(value ~/ 1000);       // 2
print((value ~/ 100) % 10); // 3
print((value % 100) ~/ 10); // 5
print(value % 10);          // 4
