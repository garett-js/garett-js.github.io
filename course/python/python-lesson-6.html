Остаток от деления. Инкремент и Декремент

Инкремент и Декремент

Одна их самых частых операций это увеличить или уменьшить значение переменной.

Это можно сделать, изменив значение переменной, и новое значение присвоить в ту же самую переменную.

Python - Изменение значения переменной
value: int = 42

value = value + 7  # 49
value = value - 7  # 42

# value = 42 + 7 = 49
# value = 49 - 7 = 42

Эти операции встречаются настолько часто, что для них придумали специальные сокращенные (комбинированные) операторы.

Python - Комбинированные операторы
value: int = 42

value += 7  # value = value + 7
value -= 7  # value = value - 7
value *= 7  # value = value * 7
value /= 7  # value = value / 7 (результат будет float)
value //= 7 # value = value // 7 (целочисленное деление)
value %= 7  # value = value % 7

Отсутствие инкремента и декремента в Python

В отличие от многих других языков (включая Dart, C++, Java), в Python нет операторов инкремента (++) и декремента (--).

Вместо этого для увеличения или уменьшения переменной на единицу используются стандартные комбинированные операторы:

Python - Эмуляция инкремента и декремента
value = 10
# Увеличение на 1
value += 1  # Теперь value равно 11

# Уменьшение на 1
value -= 1  # Теперь value равно 10

# Записи value++ или value-- вызовут синтаксическую ошибку в Python!

Поскольку в Python нет операторов ++ и --, то и понятий префиксной и постфиксной форм для них не существует. Это избавляет от целого класса потенциальных ошибок и делает код более предсказуемым.

Сценарии использования оператора остатка от деления (%)

Проверка на целочисленное деление

Очень часто необходимо проверить, делится ли одно число на другое без остатка. В этом помогает операция остатка от деления (%). Если остаток от деления равен 0, то число делится нацело.

Например, проверим, кратно ли число пяти.

Python - Проверка кратности 5
value: int = 25

if value % 5 == 0:
  print(f"Число {value} кратно 5")
else:
  print(f"Число {value} не делится на 5 нацело")

Например, проверим, делится ли число на 5 и на 3 одновременно.

Python - Проверка кратности 5 и 3
value: int = 15
 
if (value % 5 == 0) and (value % 3 == 0):
  print(f"Число {value} кратно 5 и 3 одновременно")
else:
  print(f"Число {value} не делится на 5 и 3 одновременно")

Например, проверим, является ли число чётным (т.е. делится ли оно на 2 без остатка).

Python - Проверка на четность
value: int = 42

if value % 2 == 0:
  print(f"Число {value} чётное")
else:
  print(f"Число {value} нечётное")

Получение числовых разрядов

С помощью операторов % и // (целочисленное деление) можно легко получать отдельные цифры числа.

Python - Получение разрядов числа
value: int = 2354

# Получить каждую цифру по отдельности
# Для целочисленного деления в Python используется оператор //

thousands = value // 1000      # 2354 // 1000 = 2
hundreds = (value // 100) % 10 # (2354 // 100) = 23; 23 % 10 = 3
tens = (value // 10) % 10      # (2354 // 10) = 235; 235 % 10 = 5
units = value % 10             # 2354 % 10 = 4

print(f"Тысячи: {thousands}")    # 2
print(f"Сотни: {hundreds}")      # 3
print(f"Десятки: {tens}")        # 5
print(f"Единицы: {units}")      # 4
