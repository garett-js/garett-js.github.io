Задача № 1: Приветствие

Уровень сложности: ⭐

Создайте переменную для хранения имени пользователя, а затем используйте её для вывода приветствия. Используйте функцию input() для получения имени.

Примерный вывод
Введите ваше имя: Гаретт
Привет, Гаретт!
Решение
# Запрашиваем имя пользователя
user_name = input("Введите ваше имя: ")

# Выводим приветствие, используя f-строку
print(f"Привет, {user_name}!")

Задача № 2: Площадь треугольника

Уровень сложности: ⭐

Используя переменные и шаблонные строки (f-строки), найдите площадь треугольника. Задайте основание и высоту в переменных.

  • Основание треугольника: 5
  • Высота треугольника: 10
Ожидаемый вывод
Площадь треугольника с основанием 5 и высотой 10 равна 25.0

Подсказка: Формула площади треугольника: (основание * высота) / 2.

Решение
# Задаем основание и высоту
base = 5
height = 10

# Рассчитываем площадь
area = (base * height) / 2

# Выводим результат
print(f"Площадь треугольника с основанием {base} и высотой {height} равна {area}")

Задача № 3: Обмен значений переменных

Уровень сложности: ⭐

Создайте две переменные a и b, присвойте им начальные значения. Затем поменяйте их значения местами, используя "НЕ питонический" и "Питонический" способ, и выведите результат.

Примерный вывод
До обмена: a = 15, b = 30
После обмена: a = 30, b = 15
Решение
# Исходные значения
a = 15
b = 30
print(f"До обмена: a = {a}, b = {b}")

# Способ 1: "Не питонический" (через временную переменную)
temp = a
a = b
b = temp
print(f"После обмена (не питонический): a = {a}, b = {b}")

# Вернем значения для второго способа
a = 15
b = 30

# Способ 2: "Питонический" (множественное присваивание)
a, b = b, a
print(f"После обмена (питонический): a = {a}, b = {b}")

Задача № 4: Средний балл оценок

Уровень сложности: ⭐

Создайте три переменные для хранения оценок по разным предметам (например, математика, физика, информатика). Рассчитайте и выведите среднюю оценку, используя f-строку для форматирования вывода.

Примерный вывод
Оценки: Математика=5, Физика=4, Информатика=5
Средний балл: 4.66666...
Решение
# Оценки по предметам
math_grade = 5
physics_grade = 4
informatics_grade = 5

# Расчет среднего балла
average_grade = (math_grade + physics_grade + informatics_grade) / 3

# Вывод результатов
print(f"Оценки: Математика={math_grade}, Физика={physics_grade}, Информатика={informatics_grade}")
print(f"Средний балл: {average_grade}")

Задача № 5 — Работа с переменными

Уровень сложности: ⭐

Объявите три переменные типа int и присвойте первой числовое значение, вторая переменная равна первой переменной увеличенной на 3, а третья переменная равна сумме первых двух.

Подсказка: Используйте арифметические операции + для сложения.

Решение
# Объявляем первую переменную
var1 = 10

# Вторая переменная на 3 больше первой
var2 = var1 + 3

# Третья переменная - сумма первых двух
var3 = var1 + var2

# Выводим значения всех переменных
print(f"Первая переменная: {var1}")
print(f"Вторая переменная: {var2}")
print(f"Третья переменная: {var3}")

Задача № 6 — Обмен значений трех переменных

Уровень сложности: ⭐⭐

Составить программу обмена значениями трех переменных величин a, b, c по следующей схеме:

  • а) b присвоить значение c, a присвоить значение b, c присвоить значение a;
  • б) b присвоить значение a, c присвоить значение b, a присвоить значение c.

Подсказка: Используйте дополнительные переменные для временного хранения значений. Выведите значения переменных до и после обмена.

Решение
# Исходные значения
a = 10
b = 20
c = 30
print(f"Исходные значения: a={a}, b={b}, c={c}\n")

# --- Схема А ---
# b -> c, a -> b, c -> a (новые b,a,c получают значения старых c,b,a)
# Чтобы сохранить исходные значения, используем временные переменные
old_a = a
old_b = b
old_c = c

a_a = old_b
b_a = old_c
c_a = old_a
print(f"Схема А (через доп. переменные): a={a_a}, b={b_a}, c={c_a}")

# Питонический способ для схемы А
a_py, b_py, c_py = 10, 20, 30
a_py, b_py, c_py = b_py, c_py, a_py
print(f"Схема А (питонический способ):    a={a_py}, b={b_py}, c={c_py}\n")


# --- Схема Б ---
# b -> a, c -> b, a -> c (новые b,c,a получают значения старых a,b,c)
old_a = a
old_b = b
old_c = c

b_b = old_a
c_b = old_b
a_b = old_c
print(f"Схема Б (через доп. переменные): a={a_b}, b={b_b}, c={c_b}")

# Питонический способ для схемы Б
a_py, b_py, c_py = 10, 20, 30
a_py, b_py, c_py = c_py, a_py, b_py
print(f"Схема Б (питонический способ):    a={a_py}, b={b_py}, c={c_py}")

Задача № 7 — Возведение в степень через умножение

Уровень сложности: ⭐⭐

Дано вещественное число a. Пользуясь только операцией умножения, получите:

  • а) a⁴ (а в степени 4) за две операции;
  • б) a⁶ за три операции;
  • в) a⁷ за четыре операции;

Подсказка: Используйте промежуточные переменные для хранения результатов умножения. Например, для a⁴: сначала a², затем (a²)².

Решение
a = 2
print(f"Исходное число a = {a}\n")

# а) a^4 за 2 операции
# a^2 = a * a
# a^4 = a^2 * a^2
a2 = a * a      # Операция 1
a4 = a2 * a2    # Операция 2
print(f"a^4 = {a4} (за 2 операции)")

# б) a^6 за 3 операции
# a^2 = a * a
# a^3 = a^2 * a
# a^6 = a^3 * a^3
a2 = a * a      # Операция 1
a3 = a2 * a     # Операция 2
a6 = a3 * a3    # Операция 3
print(f"a^6 = {a6} (за 3 операции)")

# в) a^7 за 4 операции
# a^2 = a * a
# a^4 = a^2 * a^2
# a^6 = a^4 * a^2
# a^7 = a^6 * a
a2 = a * a      # Операция 1
a4 = a2 * a2    # Операция 2
a6_v2 = a4 * a2 # Операция 3
a7 = a6_v2 * a  # Операция 4
print(f"a^7 = {a7} (за 4 операции)")
