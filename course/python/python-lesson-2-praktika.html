Практические задания - Урок 2: Переменные и константы

Инструкция: Решите все задания по порядку. Создайте отдельный файл .py для каждой задачи или решите все в одном файле, разделяя задачи комментариями.

Задача № 1: Приветствие

Уровень сложности: ⭐

Создайте переменную для хранения имени пользователя, а затем используйте её для вывода приветствия. Используйте функцию input() для получения имени.

Примерный вывод
Введите ваше имя: Гаретт
Привет, Гаретт!

Задача № 2: Площадь треугольника

Уровень сложности: ⭐

Используя переменные и шаблонные строки (f-строки), найдите площадь треугольника. Задайте основание и высоту в переменных.

  • Основание треугольника: 5
  • Высота треугольника: 10
Ожидаемый вывод
Площадь треугольника с основанием 5 и высотой 10 равна 25.0

Подсказка: Формула площади треугольника: (основание * высота) / 2.

Задача № 3: Обмен значений переменных

Уровень сложности: ⭐

Создайте две переменные a и b, присвойте им начальные значения. Затем поменяйте их значения местами, используя "НЕ питонический" и "Питонический" способ, и выведите результат.

Примерный вывод
До обмена: a = 15, b = 30
После обмена: a = 30, b = 15

Задача № 4: Средний балл оценок

Уровень сложности: ⭐

Создайте три переменные для хранения оценок по разным предметам (например, математика, физика, информатика). Рассчитайте и выведите среднюю оценку, используя f-строку для форматирования вывода.

Примерный вывод
Оценки: Математика=5, Физика=4, Информатика=5
Средний балл: 4.66666...

Задача № 5: Характеристики персонажа

Уровень сложности: ⭐⭐

Создайте программу, которая запрашивает у пользователя имя, класс и уровень игрового персонажа. Затем рассчитайте и выведите его базовые характеристики.

  1. Используйте input(), чтобы получить имя, класс и уровень.
  2. Уровень нужно преобразовать в число.
  3. Определите константу BASE_HEALTH со значением 100.
  4. Рассчитайте здоровье персонажа по формуле: здоровье = BASE_HEALTH + уровень * 10.
  5. Выведите итоговую информацию о персонаже в виде анкеты с помощью f-строки.
Примерный вывод
Введите имя персонажа: Артас
Введите класс персонажа: Паладин
Введите уровень: 12
--- Анкета Персонажа ---
Имя: Артас
Класс: Паладин
Уровень: 12
Здоровье: 220
-----------------------

Задача № 6: Калькулятор лута

Уровень сложности: ⭐⭐

Игрок нашел сокровище! Напишите программу, которая подсчитывает общую стоимость добычи в медных монетах.

  1. Используйте input(), чтобы спросить, сколько золотых, серебряных и медных монет нашел игрок.
  2. Определите константы для обменного курса: SILVER_PER_GOLD = 10, COPPER_PER_SILVER = 100.
  3. Рассчитайте общую стоимость всей добычи в медных монетах.
  4. Выведите результат с поясняющим сообщением.
Примерный вывод
Сколько золотых монет найдено? 3
Сколько серебряных монет найдено? 15
Сколько медных монет найдено? 75
Общая стоимость добычи: 4575 медных монет.

Задача № 7: Расчет урона в бою

Уровень сложности: ⭐⭐

Создайте программу, имитирующую расчет урона в ролевой игре. У героя есть базовый урон, а у монстра — броня.

  1. С помощью множественного присваивания задайте базовый урон героя и броню монстра (например, 25 и 10).
  2. Определите константу для множителя критического удара CRIT_MULTIPLIER = 1.5.
  3. Рассчитайте урон от критического удара.
  4. Рассчитайте итоговый урон, который получит монстр (урон от крита минус броня).
  5. Выведите весь лог боя (информацию о базовых параметрах, расчете крита и итоговом уроне) с помощью нескольких print() и f-строк.
Примерный вывод
Герой атакует с базовым уроном 25.
У монстра 10 единиц брони.
Критический удар! Урон увеличен в 1.5 раз и составляет 37.5.
Монстр получает 27.5 урона.
