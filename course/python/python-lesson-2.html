Переменные и константы в Python

Зачем нужны переменные

На прошлом занятии мы практиковались с арифметическими операциями. Но любая операция с числами не имеет смысла, если невозможно получить результат в виде значения.

Для хранения любых данных, в том числе и результата арифметических операций, в программировании используют переменные.

Переменная

Переменная - это область памяти, где можно хранить и изменять данные.

У переменной всегда есть название (имя переменной)

Переменная как коробка

Можно представить переменную как коробку, внутри которой находится значение.

У этой коробки есть ярлык - имя переменной. Обращаясь к коробке по её имени, вы получаете доступ к хранящемуся в ней значению.

Эта схема иллюстрирует, как переменная может хранить данные, подобно тому, как коробка может содержать какой-то предмет. На коробке написано Имя. Снаружи коробки находится метка Данные, указывающая на значение. Стрелка показывает, что это значение помещается внутрь коробки.

Зачем нужно использовать коробку с именем, чтобы хранить значение, а не просто использовать значения напрямую?
Python
print(42) # Целое число 42
print(42 + 7) # Сложим два целых числа
print("Привет, Python") # Напечатаем строку

Объявление переменной в Python

В Python переменные объявляются в момент присваивания им значения.

Тип данных указывать не обязательно - Python определит его автоматически.

Этот процесс называется динамической типизацией.

Но я рекомендую в будущем указывать Тип данных для того, чтобы было понятно, какой тип данных хранится в переменной.

Оператор присваивания

В объявлении переменной после имени переменной следует символ равно =

Этот символ называется оператором присваивания.

Оператор присваивания сохраняет значение в переменную.

Python
count: int = 2 # Объявляем целочисленную переменную
print(count) # Вызываем переменную

Инициализация переменной

Фраза: переменная count инициализирована значением 2 означает, что 2 это первое значение, сохраненное в переменной при её создании.

В переменную значения присваиваются справа налево

Множественное присваивание и обмен значениями

Python позволяет делать некоторые вещи очень изящно. Давайте посмотрим на один из таких "фокусов", который упрощает код.

Вы можете присвоить значения сразу нескольким переменным в одной строке, экономя место и делая код чище.

Python - Множественное присваивание
# Раскладываем "значения" по "коробкам" одним движением
name, city, age = "Алиса", "Москва", 25

print(f"Имя: {name}")
print(f"Город: {city}")
print(f"Возраст: {age}")

Магия Python: меняем значения местами (Swap)

А теперь задачка: у вас есть две переменные, скажем, a и b. Как поменять их значения местами?

В большинстве языков для этого нужна третья, временная переменная. Это как иметь две чашки с разными напитками и хотеть их поменять — нужна третья, пустая чашка.

Python - Классический способ обмена
a = 5
b = 10

# Нужна временная "чашка"
temp = a
a = b
b = temp

print(f"a теперь: {a}") // a теперь: 10
print(f"b теперь: {b}") // b теперь: 5

Фишка Python

Но Python — язык с изюминкой! Он позволяет сделать это элегантно и без лишних переменных. Это одна из маленьких "фишек", которую любят разработчики.

Python - "Питонический" способ обмена (Swap)
a = 5
b = 10

# Магия Python!
a, b = b, a

print(f"a теперь: {a}") // a теперь: 10
print(f"b теперь: {b}") // b теперь: 5

Как это работает? На самом деле, это просто более хитрое использование множественного присваивания. Python сначала "вычисляет" значения справа (берёт текущие значения b и a), а потом присваивает их переменным слева. Всё происходит как бы одновременно, поэтому временная переменная не нужна.

Переменные и оперативная память

Данные в памяти

Для того чтобы программа могла воспользоваться данными, эти данные нужно где-то хранить.

Хранятся они в специальных ячейках памяти (оперативной памяти)

... не в красных стаканчиках конечно 😅

Когда мы в программе пишем значение просто 42 или Привет Python, то не понятно что это за данные, где они находятся, откуда взялись и как с ними работать потом.

Это какие-то магические значения

Переменные помогают контролировать данные, с помощью переменной мы можем сохранить данные в оперативной памяти и потом ими воспользоваться

Чисто технически в самой переменной не находятся данные, в ней находится ссылка на область памяти, где хранятся данные.

Python
day_of_week = "Понедельник"
print(day_of_week) # "Понедельник"

day_of_week = "Вторник" # Обновляем значение переменной
print(day_of_week) # "Вторник"

Старый объект "Понедельник" остается в памяти, но на него больше не указывает ни один стикер. Через некоторое время "сборщик мусора" Python увидит, что объект больше не используется, и удалит его, освободив память.

Ключевой вывод:

Мы не изменили старый объект. Мы заставили имя ссылаться на новый объект.

Неизменяемые (immutable) типы данных: int, float, str, tuple. С ними всегда создается новый объект, а имя-ссылка перенаправляется.

В Python переменные — это имена-ярлыки, которые ссылаются на объекты в памяти.

Когда вы присваиваете новое значение переменной неизменяемого типа (как строка), вы не меняете старый объект, а создаете новый и перенаправляете имя на него.

Данные в ячейках памяти

Ячейки памяти

Технически хранение переменных можно представить как ячейки памяти (оперативной) в виде таблицы.

Столбцы и строки этой таблицы пронумерованы, т.е. представляют из себя адреса, и вот по этому адресу в память можно записать какие-то данные.

Константы

Константы

Это переменные значения которых нельзя изменить

В Python нет констант, как в более серьезных и строгих языках программирования. Однако, по соглашению (PEP 8), переменные, которые не должны изменяться, именуются заглавными буквами в стиле UPPER_SNAKE_CASE.

Это не мешает программисту изменить значение такой "константы", но служит сигналом для других разработчиков, что это значение менять не предполагается и не желательно 😞

Python
PI = 3.14 # Объявляем "константу"
# PI = 4 # Так делать не рекомендуется!
print(PI)

# Пример "константы"
PLAYER_MAX_HEALTH = 100

Шаблонные строки (f-строки)

Рассмотрим пример, когда вывод простого числа не очень информативен. Было бы полезнее вывести сообщение, поясняющее, что означает число

У вас 2 непрочитанных сообщения
Python
count = 2
print("У вас count непрочитанных сообщений")

# У вас count непрочитанных сообщений

Можно взять и объединить несколько строчек друг с другом и с переменной c помощью оператора +. Но … код получается плохо читаемый и многословный 😥

Python
count = 2
    
# Такой подход называется КОНКАТЕНАЦИЯ строк
print("У вас " + str(count) + " непрочитанных сообщений")

Обратите внимание, что мы преобразовали число count в строку с помощью str(), потому что в Python нельзя напрямую складывать строки и числа.

f-строки

Чтобы это безобразие исправить нужно использовать f-строки (форматированные строковые литералы).

Поставьте букву f перед открывающей кавычкой строки, а имя переменной заключите в фигурные скобки {}.

Python
count = 2
    
# Такой подход называется ИНТЕРПОЛЯЦИЯ строк
print(f"У вас {count} непрочитанных сообщений")
    
# У вас 2 непрочитанных сообщения

Если нужно выполнить более сложное выражение внутри f-строки, его также можно поместить в фигурные скобки.

Python
unread_count = 5
read_count = 100
    
print(f"Всего у вас {unread_count + read_count} сообщений")
    
# Всего у вас 105 сообщений

Ввод и вывод данных через консоль

Быстрый I/O в Python

  • print() — основной способ вывода данных в консоль.
  • input() — основной способ получения строковых данных от пользователя.

Эти функции просты, кроссплатформенны и не требуют дополнительных библиотек, поэтому идеально подходят для учебных примеров и задач.

Расширенные возможности print()

Python
print("Hello", "world", sep="--", end="!\n")  # Hello--world!
  • sep — разделитель между аргументами (по умолчанию пробел).
  • end — символ(ы), добавляемые в конец (по умолчанию \n).
  • flush=True — принудительно сбросить буфер (актуально при долгих вычислениях).

Вызов команды print() с пустыми скобками делает перевод строки

print("Hello")
print()
print("World")
Будет 
Hello
    
World

Функция input()

Программы могут не только выводить заранее известный текст, но и работать с данными, которые пользователь вводит во время выполнения.

В Python для этого используется команда input(). Она позволяет программе считывать введённые данные, чтобы затем их использовать.

Программа будет ждать от пользователя ввода данных. Ввод данных реализуется с помощью команды input()

Python
name = input("Введите имя героя: ")
print(f"Привет, {name}!")

Преобразование типов

Так как input() всегда возвращает строку, для работы с числами необходимо ЯВНО преобразовать тип:

Python
level_str = input("Введите уровень персонажа: ") # '12'
level = int(level_str) # 12 (int)
level += 2 # 14 (int)

# или короче:
level = int(input("Введите уровень персонажа: "))
